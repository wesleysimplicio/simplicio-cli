# react-vite

TypeScript + React 18 + Vite scaffold for single-page applications with a fast
development loop and Vitest component tests.

## When to use this stack

- Frontend-only app or dashboard
- SPA with client-side routing or API consumption
- React is desired but SSR is not needed

## Layout produced

```
<project_name>/
|-- index.html
|-- src/App.tsx
|-- src/App.test.tsx
|-- package.json
`-- README.md
```

## Verify loop

- `install`: `npm install`
- `test`: `npm test`
- `lint`: `npm run lint`
- `build`: `npm run build`
