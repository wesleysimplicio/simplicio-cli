# ts-nestjs

TypeScript + NestJS + Fastify scaffold for structured Node APIs with dependency
injection, modules, controllers, and service-oriented boundaries.

## When to use this stack

- Medium-sized backend API in TypeScript
- Team wants explicit modules and dependency injection
- Fastify adapter is preferred over Express

## Layout produced

```
<project_name>/
|-- src/app.controller.ts
|-- src/app.module.ts
|-- src/main.ts
|-- package.json
`-- README.md
```

## Verify loop

- `install`: `npm install`
- `test`: `npm test`
- `lint`: `npm run lint`
- `build`: `npm run build`
