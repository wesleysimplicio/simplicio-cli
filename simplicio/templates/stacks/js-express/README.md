# js-express

JavaScript + Express 4 scaffold for small Node 20 JSON APIs.

## When to use this stack

- REST API in plain JavaScript
- Node service where TypeScript is not required
- Fast local iteration with `node --test`

## Layout produced

```
<project_name>/
|-- src/app.js
|-- src/server.js
|-- tests/health.test.js
|-- package.json
`-- README.md
```

## Verify loop

- `install`: `npm install`
- `test`: `npm test`
- `lint`: `npm run lint`
