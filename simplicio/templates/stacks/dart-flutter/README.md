# dart-flutter

Dart + Flutter scaffold for cross-platform mobile and desktop apps.

## When to use this stack

- Cross-platform mobile app
- UI and state are Flutter-first
- One codebase should target Android, iOS, and desktop

## Layout produced

```
<project_name>/
|-- pubspec.yaml
|-- lib/main.dart
|-- test/widget_test.dart
`-- README.md
```

## Verify loop

- `install`: `flutter pub get`
- `test`: `flutter test`
- `lint`: `flutter analyze`
